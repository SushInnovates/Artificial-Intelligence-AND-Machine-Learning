# Day 10 - Literals in Python

## Definition
A literal is raw data assigned to a variable.

Example:
x = 10

Here, 10 is the literal.

## Types of Literals in Python

1. Numeric Literals
2. String Literals
3. Boolean Literals
4. Special Literals

## Examples

Numeric:
- Binary → 0b1010
- Octal → 0o310
- Hexadecimal → 0x12c

Boolean:
- True
- False

Special:
- None
